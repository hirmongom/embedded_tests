Detailed Description

Header file for GPIO peripheral driver.

This file provides the necessary definitions and functions for configuring and controlling GPIO pins on the STM32F10RB microcontroller.

See also
RM0401 Reference Manual, Page 135 for more information on GPIO configuration.
